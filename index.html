About

I'm the consultant behind A2I Softech, an independent IT practice based in the North West of England. Over the past twenty years I've worked as a hands-on developer, architect and consultant for organisations ranging from national vehicle-buying networks to life-sciences platforms and logistics operators — usually brought in to solve a specific, business-critical problem rather than to sit in meetings about one.

I still write code every day. I care about systems that are simple to operate, easy to hand over, and built to survive contact with real users — not just architecture diagrams. Clients bring me in for greenfield builds, for rescuing projects that have stalled, and for the kind of independent technical opinion that's hard to get from inside a single vendor or team.

Selected work

Case studies are described in general terms to respect client confidentiality.

Logistics & Workforce Management

Multi-Depot Attendance & Workforce Reporting Platform

Designed and built a clocking-in system spanning Raspberry Pi devices at depot entrances and a companion Android app with RFID scanning, capturing employee attendance with photo verification and feeding a .NET Core / Azure Functions backend.

Outcome: replaced paper-based timesheets across multiple depots, automated shift and payroll reporting, and gave management real-time visibility of attendance and exceptions.

  • ASP.NET Core
  • Azure Functions
  • Angular
  • Xamarin
  • SQL Server

Life Sciences & Data Platforms

Unified Sample Search & Ordering Platform

Implemented single sign-on across internal and third-party modules — catalogue, analytics and commerce — for a scientific supplies platform, and designed the secure API layer consumed by both web and desktop clients.

Outcome: unified a fragmented multi-vendor ecosystem into one coherent experience, cutting the steps required to search for and purchase biological samples.

  • ASP.NET Core
  • Identity Server
  • WPF
  • Web API
  • Angular

Automotive & Consumer Finance

Guided Vehicle Purchase & Instant Payment System

Built a 20-step guided purchase wizard integrating third-party vehicle-data and registration-check APIs, alongside a finance module wired into a major bank's payment API for same-day settlement.

Outcome: automated end-to-end payment processing for vehicle purchases and materially reduced manual finance administration.

  • ASP.NET MVC
  • Web API
  • AngularJS
  • WebSockets
  • SQL Server

Travel & Accessibility

Accessible Rewards Module for a Major Booking Platform

Delivered a customer-facing rewards module to WCAG 2.1 AA accessibility standards, working alongside a dedicated accessibility specialist through design, build and client handover.

Outcome: passed formal accessibility audit and was handed over directly to the client's in-house team with no post-handover defects reported.

  • ASP.NET MVC
  • AngularJS
  • SASS
  • WCAG AA
